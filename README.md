# EZ Labs Frontend

A responsive React.js website for EZ Labs with an integrated contact form.

## Features
- Responsive design (mobile, tablet, desktop)
- Contact form with validation
- API integration
- Modern UI with animations

## Tech Stack
- React.js + Vite
- CSS3
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```

## API Endpoint
Contact form submits to: `https://vernanbackend.ezlab.in/api/contact-us`
